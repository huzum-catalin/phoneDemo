Assessment instructions
Please create a JavaScript app to display all the phones described in feed.json.

The app should:

1.       Display all phones in a gallery layout.
2.       Show details of an individual phone when it is selected.


To complete this test please
- add in extra functionality and features, show us your imagination.
- feel free to use any libraries or frameworks that you like.
- treat this as if it should be production ready.
- think about how this might need to scale.
- pay attention to performance.
